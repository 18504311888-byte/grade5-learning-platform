# 学习平台审计与改进建议

审计日期：2026-06-17

## 知识正确性

- 数学：已有 `五年级下册数学-Codex自学计划.md`、`问题库`、`30 学生模拟测试`，核心知识点覆盖较完整。
- 英语：已核验外研社五下目录和 20 个 Unit 课文正文；当前题库只使用已核验的课文句型、关键词和基础主题。
- 英语仍需继续核验：完整单词表、Activity 原题、Review Module 具体练习、音频。

## Agent / Skills 规则

已增加教材核验规则：

- 不能凭记忆或模型猜测教材结构。
- 必须先看实物教材、官方资料或可公开读取的正文。
- 发现错误版本必须先承认并修正。
- 每次教材更新必须记录来源。

建议后续增加：

1. `教材来源清单.md`：每个知识点对应来源链接。
2. `题目来源字段`：每道英语题记录 Unit、证据页、是否改编。
3. `人工复核状态`：未核验题不允许进入正式练习。

## 程序逻辑问题

本次修复：

- 删除旧英语题库残留，避免 My Day 等错误教材内容混入。
- 修复英语默认主题仍指向 My Day 的问题。
- 删除重复 `askAiHint` 函数，避免旧 API 覆盖新 API。
- 更新 Service Worker 缓存版本，避免浏览器继续使用旧教材内容。

## 积分机制

原积分机制只按分数加分，孩子容易只刷题。

本次增加：

- 每日第一次完成练习额外 +3 分。
- 已掌握知识点获得徽章。
- 20 / 50 / 100 / 180 / 300 分里程碑记录。
- 每日任务栏显示“每日+3分已获得”。

后续可以继续增加：

- 连续学习 3 天、7 天奖励。
- 连续正确 3 题奖励星星。
- 错题复习完成奖励。
- 家长确认奖励卡片。
